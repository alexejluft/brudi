# i18n Base Structure

All base translation files follow this exact key structure.
Agent copies the relevant language file, then adds project-specific keys.

## Categories

```
common.nav          — Navigation links
common.footer       — Footer text, links, copyright
common.cta          — Call-to-action buttons
common.aria         — Accessibility labels (screen readers)
common.theme        — Dark/Light mode labels
common.locale       — Language switcher labels
common.loading      — Loading/skeleton states
common.errors       — Error messages (404, 500, generic)
common.forms        — Form labels, placeholders, validation
common.faq          — FAQ section labels
common.cookies      — Cookie banner text
common.social       — Social media labels
legal.impressum     — Impressum section headings
legal.datenschutz   — Privacy policy section headings
legal.placeholder   — Placeholder notices for missing data
```
