RULE SUMMARY
================================================================================

Rule 1: no-transition-all
  • Prevents: transition: all (Tailwind + inline styles)
  • Severity: Error (blocks commits)
  • Reason: 30-50% slower, animates unintended properties
  • Implementation: Detects in JSX attributes, style objects, CSS

Rule 2: no-gsap-from-in-react
  • Prevents: gsap.from() in useEffect
  • Severity: Error (blocks commits)
  • Reason: Causes FOUC (Flash of Unstyled Content)
  • Implementation: Detects GSAP function calls, timeline calls

Rule 3: scrolltrigger-cleanup-required
  • Prevents: ScrollTrigger without cleanup in useEffect return
  • Severity: Warning (doesn't block, but should be fixed)
  • Reason: Memory leaks, duplicate triggers on navigation
  • Implementation: Analyzes useEffect cleanup function

Rule 4: no-layout-animation
  • Prevents: Animating margin, width, height, padding, position
  • Severity: Warning (optimization, can refactor)
  • Reason: 30-60% frame drops from layout recalculation
  • Implementation: Detects in GSAP calls and CSS transitions
