# brudi-webdev

Award-level web development skills for AI coding agents.

## Installation

```bash
git clone https://github.com/alexejluft/brudi-webdev.git ~/.openclaw/skills/brudi-webdev
```

## Skills (8 total, ~100 lines each)

| Skill | Use When |
|-------|----------|
| **building-layouts** | Creating layouts with Grid, Flexbox, Container Queries |
| **designing-for-awards** | Making visual design decisions, avoiding AI-slop |
| **animating-interfaces** | Adding GSAP, Lenis, or CSS animations |
| **developing-with-react** | Working with Server Components, hooks, state |
| **typing-with-typescript** | Writing type-safe code with utility types |
| **testing-user-interfaces** | Testing components with Vitest, Testing Library |
| **optimizing-performance** | Fixing Core Web Vitals, improving speed |
| **building-accessibly** | Ensuring WCAG compliance, keyboard navigation |

## Philosophy

- **Tiefe > Breite** — Deep understanding over shallow knowledge
- **Anti-AI-Slop** — No generic patterns
- **Evidence over claims** — TDD-tested skills
- **Token efficient** — ~100 lines per skill

See [docs/philosophy.md](docs/philosophy.md)

## What Makes This Different

Every skill is:
1. **Failure-tested** — Designed around common AI mistakes
2. **Real-world sourced** — Patterns from production projects
3. **Compact** — Dense knowledge, not verbose tutorials
4. **Opinionated** — We make choices, not suggestions
